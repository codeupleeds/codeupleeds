About

CodeUp is the adult version of Code Club, providing free and friendly adult coding tuition.

The event is an open, friendly and supportive learning environment where anyone can learn or improve their digital skills for free.

Resources

Meetup

All of the CodeUp Leeds events are organized via Meetup. If you're struggling to find information about the date, time, location or people at an event, check here first.

GitHub

Git is industry-leading version control software and is incredibly popular with professionals and hobbyists alike. CodeUp Leeds has a public GitHub repository where we encourage you to store any public projects you work on at CodeUp. We even run this web site direct from our GitHub repository, We also encourage you to sign up to GitHub so that you can use this when required.

Slack

CodeUp Leeds has a Slack workspace where you can chat, share files, ask questions and get help with problems in between Meetups. Follow this link to request an invitation. You can download Slack here.
